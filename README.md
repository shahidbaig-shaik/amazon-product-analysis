Amazon Product Analysis is a data-driven web application designed to help users explore and understand product trends, pricing patterns, and customer sentiment on Amazon. Built using Python and Streamlit, this tool provides an interactive interface for analyzing large datasets of Amazon products.
Key Objectives

Data Insights: Extract meaningful insights from product listings, including price distribution, ratings, and reviews.
Visualization: Present data through dynamic charts and graphs for easy interpretation.
AI-Powered Analysis: Incorporate natural language processing to summarize customer feedback and identify sentiment trends.
User-Friendly Interface: Enable non-technical users to interact with data effortlessly through a clean and intuitive dashboard.

Core Features

Product Search & Filtering: Quickly find products based on category, price range, and rating.
Interactive Charts: Visualize pricing trends, rating distributions, and top-performing products.
Sentiment Analysis: Understand customer opinions using AI-driven text analysis.
Chat-Based Insights: Ask questions and get instant insights from the dataset.

Technology Stack

Frontend: Streamlit for building interactive dashboards.
Backend: Python for data processing and analysis.
Deployment: Hugging Face Spaces for hosting and sharing the app online.
